# 117 Seguridad Industrial

Plataforma web para la gestión y comercialización de servicios de seguridad industrial, capacitación empresarial y equipamiento de emergencia en Costa Rica.

## Descripción

117 Seguridad Industrial es una solución digital integral que combina un sitio web público informativo con un panel administrativo privado, permitiendo a la empresa gestionar contenidos, imágenes y servicios de forma dinámica y segura.

La plataforma está diseñada para mostrar cursos de capacitación, servicios de salud ocupacional, equipamiento de emergencia y material didáctico, con una experiencia moderna y optimizada.

## Funcionalidades

### Público
- Catálogo de cursos y servicios
- Información de salud ocupacional
- Galerías y carruseles de equipamiento
- Formulario de contacto
- Integración con WhatsApp
- Tema claro y oscuro
- Analítica con Google Analytics 4

### Administrador
- Autenticación segura
- Gestión dinámica de imágenes y contenidos
- Panel administrativo protegido
- Actualización en tiempo real sin reiniciar la aplicación

## Tecnologías

- Frontend: React, TypeScript, Vite
- Backend: Node.js, Express, Prisma
- Base de datos: PostgreSQL
- Infraestructura: Docker, Cloudinary, Google Analytics 4

## Ejecución rápida


npm install
npm run dev:full
